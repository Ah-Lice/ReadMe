## 🦁 Projeto em destaque / Featured Project

### Escute as Feras — CLEF
**Plataforma web para o Clube de Leitura Escute as Feras**, um espaço literário feminino com blog, biblioteca digital clandestina, acontecimentos mensais e muito mais.

*Web platform for the Escute as Feras Literary Club — a women's literary space featuring a blog, a clandestine digital library, monthly book discussions and more.*

> Repositório privado / Private repository

**Stack utilizada / Stack used:**

| Camada | Tecnologia |
|--------|-----------|
| Frontend | Angular 19 · TypeScript · Tailwind CSS 4 · Angular Material |
| Backend | Spring Boot 4 · Java 17 · Spring Security 7 · JWT |
| Banco de dados | MySQL 8 |
| Editor de texto | Quill.js |
| Autenticação | JWT · BCrypt |

**Funcionalidades implementadas / Implemented features:**
- 🔐 Autenticação completa com JWT *(Full JWT authentication)*
- 📚 Livro do mês com comentários em thread *(Book of the month with threaded comments)*
- ✍️ Blog com editor de texto rico tipo Blogger *(Blog with rich text editor like Blogger)*
- 📖 Biblioteca clandestina com upload de arquivos epub *(Clandestine library with epub file upload)*
- 🗂️ Paginação, busca e filtros *(Pagination, search and filters)*
- 🎨 Design com paleta inspirada em pintura naïf *(Design with palette inspired by naïf painting)*
